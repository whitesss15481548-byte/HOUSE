# 凰羽選宅｜Huangyu Selections

高質感黑金風不動產品牌形象網站，使用純 HTML、CSS、JavaScript 製作，適合部署到 GitHub Pages 或任一靜態網站主機。

## 本機預覽

直接開啟 `index.html` 即可瀏覽。若想用本機伺服器預覽：

```powershell
python -m http.server 4173
```

接著開啟：

```text
http://localhost:4173
```

## 圖片替換

網站圖片集中於 `assets/images/`：

- `hero-luxury-residence.webp`：首頁大圖
- `property-museum.webp`：美術館質感景觀宅
- `property-nong16.webp`：農16公園精品宅
- `property-fengshan.webp`：鳳山捷運生活宅
- `tour-thumb-1.webp`：賞屋開箱縮圖

同資料夾保留 PNG 版本，可作為原始示意圖或社群預覽圖片。若替換圖片，建議維持相同檔名，避免修改程式碼。
